# 資産形成シミュレーター

元本、毎月積立、運用期間、想定利回りから将来の資産形成を概算する静的Webアプリです。

## 主な機能

- 積立から計算
- 目標金額から必要な毎月積立額を逆算
- 税金、手数料の簡易反映
- 利回り誤差をエラーバーで表示
- 元本、積立額合計、利息合計の積み上げ棒グラフ
- 最終的な積立金の内訳円グラフ
- 年ごとの推移テーブル

## ローカル確認

```powershell
python -m http.server 4173
```

ブラウザで開きます。

```text
http://localhost:4173/
```

## GitHub Pages

このリポジトリは単一HTMLの静的サイトなので、GitHub Pagesでそのまま公開できます。

公開設定:

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/ (root)`

## 注意

本ツールは概算シミュレーションです。投資判断、税務判断、将来の運用成果を保証するものではありません。
